# Weather App

A simple weather web application that shows current weather details for any city using the OpenWeatherMap API.

## Features

* Search weather by city name
* Shows temperature, description, humidity, and wind speed
* Weather icon display
* Dynamic background based on weather condition

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## Live Demo

[https://samara328.github.io/Weather-app/](https://samara328.github.io/Weather-app/)

## How to Use

1. Enter a city name
2. Click the **Search** button
3. View the current weather information
